# Data Folder

The raw dataset is not committed to this repository to keep the repo lightweight.

**Download it from Kaggle:**
[IT Service Ticket Classification Dataset](https://www.kaggle.com/datasets/adisongoh/it-service-ticket-classification-dataset)

Place the downloaded file as:
```
data/raw/all_tickets_processed_improved_v3.csv
```

The processed files (`tickets_cleaned.csv` and `tickets_with_priority.csv`)
are auto-generated by running the notebook end-to-end.
